Stabilizer network management module

§Design

The stabilizer network architecture supports numerous layers to permit transmission of telemetry (via MQTT), configuration of run-time settings (via MQTT + Miniconf), and data streaming over raw UDP/TCP sockets. This module encompasses the main processing routines related to Stabilizer networking operations.
