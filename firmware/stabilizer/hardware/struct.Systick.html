impl Systick

source

pub fn start(systick: SYST, sysclk: u32)

Starts the Monotonic.

The sysclk parameter is the speed at which SysTick runs at. This value should come from the clock generation function of the used HAL.

Panics if it is impossible to achieve the desired monotonic tick rate based on the given sysclk parameter. If that happens, adjust the desired monotonic tick rate.

This method must be called only once.
